# JavaScript 脚本

AkariBot 提供了对 JavaScript 的支持，你可以通过脚本注册**命令**/**监听器**/**调度任务**/**变量**来实现自定义功能。

本篇教程主要讲述插件内置脚本的使用方法。如需了解如何编写自定义脚本，请移步 **进阶教程 - JavaScript 脚本**。

## 快速开始

脚本文件位于 `plugins/AkariBot/script` 路径下，将会在插件启用时自动加载。

部分脚本在使用前可能需要进行配置，修改后重载插件即可生效。

## 脚本列表

| 脚本                 | 说明         | 作者 |
|--------------------|------------|----|
| BotGroupServerChat | Bot 群服消息互通 | 季楠 |
| BotPerformCommand  | Bot 执行命令   | 季楠 |
| BotPlayerList      | Bot 玩家列表命令 | 季楠 |
