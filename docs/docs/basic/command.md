# 插件命令

AkariBot 注册了大量命令，你可以通过 `/akaribot` 或 `/abot` 查看命令帮助。

## 命令列表

| 命令                 | 描述   |
|--------------------|------|
| `/akaribot reload` | 重载插件 |

## 权限列表

| 权限                            | 描述     | 默认 |
|-------------------------------|--------|----|
| `AkariBot.command.reload.use` | 重载命令权限 | OP |
